# The Corrupt Realm

## Information
This is a singleplayer level for Quake. The main objective is to survive and escape the area. The theme is rather dark, and it should invoke a sense of anxiety and tension, rather than fear. More info for this project in the Manifesto. If you have played the map, and would like to leave feedback, please [follow this link](https://forms.office.com/Pages/ResponsePage.aspx?id=iPI-JJl3_E6v9f3k4_HJjYOVj0tBycpFoOhGAOvdA1FURFc0VTVYUzVIQlZTQlBXNVFQQ0lNQjdVMS4u)
